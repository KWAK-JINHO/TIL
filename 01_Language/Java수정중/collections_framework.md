## List 인터페이스

중복을 허용, 순서를 유지

### 특징

- 불공변성; 부모 자식간의 상속관계 없음
- 크기가 가변이다.

#### vs Array

- 배열은 공변성을 가진다.

공변성이란 두 타입간의 상속관계(B extends A)가 배열 타입에도 그대로 적용되어 B[ ]배열을 A[ ] 변수에 할당할 수 있게 허용하는 특성.
(자식 배열이 부모 타입을 할당해서 만들면 컴파일타임에는 괜찮지만 런타임에 자식 타입의 변수 넣으려고 하면 ArrayStoreException 발생)

- 배열은 크기가 고정적(크키를 바꾸려면 재할당 해야한다.)

- 배열은 제네릭 타입을 요소로 직접 사용하여 생성하는것이 불가
  (배열은 런타임에 타입을 검사하지만 제네릭은 컴파일에 타입을 체크하고 타입을 소거하기 때문에 안전성보장이 불가하다.)
