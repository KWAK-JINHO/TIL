# 자료구조란?

데이터를 효율적으로 저장하기 위한 구조화 방식. 자료구조는 배열 + 링크드리스트의 조합이다.

## 선형 자료구조 (Linear Data Structures)

1. 배열

- 크기가 고정되어 있고, 인덱스로 즉시 접근 가능 O(1)

2. 동적 배열

- 크기가 자동으로 늘어나는 배열. 가장 많이 사용.

3. 스택

- LIFO(후입선출). 되돌리기, 괄호 검사등에 사용

4. 큐

- FIFO(선입선출). BFS 알고리즘에 사용.

5. 덱

- 앞뒤 양쪽에서 넣고 빼기 가능. 스택과 큐의 기능을 모두 가진다.

## 비선형 자료구조 (Deque - Double Ended Queue)

1. 해시 맵

- key-value 쌍으로 저장하거나 중복을 제거. 탐색 속도가 가장 빠르다 O(1)

2. 우선순위 큐

- 들어온 순서와 상관없이 우선순위가 높은 데이터를 먼저 꺼낸다. (최솟값/최댓값 찾기)

3. 트리 & 그래프

- 노드간의 관계를 나타낸다.
- 자바로 구현하면 직접 Node 클래스를 설계하거나, 그래프의 경우 ArrayList<Integer>[] (인접 리스트) 형태로 구현한다.
